# Gym and Pool Management System, for Database Management System lecture

## Project Overview

*This is basically a showcase of raw SQL table Creates, Triggers and Views*

A heavy database application for managing gym and pool facilities, built with PostgreSQL and C with ECPG. The system handles member registrations, check-ins/check-outs, membership tracking, and automatic calculation of usage hours and late fees.

## Features
- **Member Management** - Add new members with address information
- **Smart Check-in System** - Real-time validation for member existence and duplicate check-ins
- **Automatic Time Tracking** - Calculates gym/pool usage hours and enforces daily limits
- **Late Fee Calculation** - Automatically computes overstay fees with grace period
- **Membership Management** - Update and manage member subscription plans
- **Real-time Monitoring** - View currently checked-in members and their session duration
